# esx_races
## Requirements
 - [esx_policejob](https://github.com/ESX-Org/esx_policejob)

## Download & Installation

### Manually
- Download https://github.com/changeME
- Put it in the `[esx]` directory

## Installation
- Import `esx_races.sql` in your database
- Add this in your `server.cfg`:

```
start esx_races
```
